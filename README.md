##–õ–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏: —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ
  –†–µ–∑—É–ª—å—Ç–∞—Ç –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è ‚Äî –ª–æ–≥–∏—á–µ—Å–∫–æ–µ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ. –õ–æ–≥–∏—á–µ—Å–∫–∏–º–∏ –Ω–∞–∑—ã–≤–∞—é—Ç
  –≤—ã—Ä–∞–∂–µ–Ω–∏—è, —Ä–µ–∑—É–ª—å—Ç–∞—Ç –∫–æ—Ç–æ—Ä—ã—Ö —Ä–∞–≤–µ–Ω ```true``` –∏–ª–∏ ```false``` .
  
  –û–ø–µ—Ä–∞—Ç–æ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è —Ü–µ–ª—ã—Ö —á–∏—Å–µ–ª:

  * ```>``` (–±–æ–ª—å—à–µ),

  –¶–µ–ª—ã–µ —á–∏—Å–ª–∞ –º–æ–∂–Ω–æ —Å—Ä–∞–≤–Ω–∏–≤–∞—Ç—å —Å –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω—ã–º–∏. –ü–µ—Ä–µ–¥ —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ–º —Ü–µ–ª–æ–µ
  —á–∏—Å–ª–æ –Ω–µ—è–≤–Ω–æ –±—É–¥–µ—Ç –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–æ –≤ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–µ–µ –µ–º—É –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω–æ–µ:

  –°—Ç—Ä–æ–∫–∏ —Å—Ä–∞–≤–Ω–∏–≤–∞—é—Ç—Å—è –ª–µ–∫—Å–∏–∫–æ–≥—Ä–∞—Ñ–∏—á–µ—Å–∫–∏ ‚Äî —Å—Ç—Ä–æ–∫–∞ ```a``` –º–µ–Ω—å—à–µ —Å—Ç—Ä–æ–∫–∏ ```b``` , –ø–æ—Ç–æ–º—É
  —á—Ç–æ –±—É–∫–≤–∞ ‚Äúf‚Äú —Å—Ç–æ–∏—Ç –≤ –∞–Ω–≥–ª–∏–π—Å–∫–æ–º –∞–ª—Ñ–∞–≤–∏—Ç–µ —Ä–∞–Ω—å—à–µ –±—É–∫–≤—ã ‚Äúw‚Äú:

## –¶–∏–∫–ª—ã while –∏ do-while
  –ß—Ç–æ–±—ã –ø—Ä–æ–≥—Ä–∞–º–º–∞ –≤—ã–ø–æ–ª–Ω—è–ª–∞ –¥–µ–π—Å—Ç–≤–∏–µ, –ø–æ–∫–∞ –µ—Å—Ç—å –Ω–µ–∫–æ—Ç–æ—Ä–æ–µ —É—Å–ª–æ–≤–∏–µ, –∏—Å–ø–æ–ª—å–∑—É—é—Ç
